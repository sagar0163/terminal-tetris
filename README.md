# Terminal Tetris 🎮

Classic Tetris game for Linux terminal - Now with enhanced features!

## Version 2.0 - Enhanced Edition

### New Features in v2.0

| Feature | Description |
|---------|-------------|
| **Ghost Piece** | See where your piece will land |
| **Hold Piece** | Save piece for later (press H) |
| **Combo System** | Multi-line clears give combo bonuses |
| **Wall Kicks** | Rotation near walls now works |
| **Speed Scaling** | Level-based difficulty progression |
| **Soft Drop** | Down arrow gives 1 point per cell |
| **Enhanced Scoring** | Up to 1200 points per clear |

### Controls

| Key | Action |
|-----|--------|
| ← → | Move left/right |
| ↓ | Soft drop |
| ↑ | Rotate |
| H | Hold piece |
| P | Pause |
| Q | Quit |

### How to Run

```bash
# Install keyboard package
pip install keyboard

# Run the game
python3 tetris.py
```

### Requirements

- Python 3.6+
- Linux terminal
- keyboard package (`pip install keyboard`)

### Game Rules

- Complete horizontal lines to clear them
- Level up every 10 lines cleared
- Speed increases with each level
- Game ends when pieces stack to top

## Features

- 🎯 7 unique tetromino shapes
- 🌈 Color-coded pieces
