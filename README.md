# Terminal Tetris

Classic Tetris game running entirely in your Linux terminal.

## Features

- 🎮 Classic Tetris gameplay
- 🎨 Color-coded tetrominoes
- 📊 Score, level, and lines tracking
- ⬇️ Hard drop shadows
- ⏸️ Pause functionality

## Requirements

```bash
pip install keyboard
```

## Controls

| Key | Action |
|-----|--------|
| ← → | Move piece |
| ↓ | Soft drop |
| ↑ | Rotate |
| P | Pause |
| Q | Quit |

## Run

```bash
python tetris.py
```
